Unter "Profil bearbeiten" kann das eigene Userprofil bearbeitet werden.

Beachte: Es können nur Mitarbeiter bearbeitet werden, die dem eingeloggten User in einer Kostenstelle (für die er verantwortlich ist) zugeordnet sind.

Unter "Mitarbeiter" können mehrere Daten eines Mitarbeiters geändert werden.

Unter "Qualifikationen" können bei allen Mitarbeitern die Qualifikationen gleichzeitig zugeordnet werden.
